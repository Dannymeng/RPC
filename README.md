### 2、银企对账工作原则 :bookmark_tabs:

*   风险导向：根据客户不同的资金风险程度， 采取差异化的对账策略，重点关注高风险客户

*   内部制衡：银企对账工作实行后台集中处理模式，对账与账务处理相分离


### 3、总分行相关部门的职责 :bookmark_tabs:

`总行运营管理部门`

*   制定我行银企对账管理规定，并组织实施

*   提出银企对账系统优化需求

*   负责对全行银企对账工作进行培训、考核、监督、检查等

`总行公司银行部`

*   负责协助总行运营管理部制定银企对账管理相关制度，配合运营管理部组织实施，负责公司网银等渠道相关功能建设工作

`总行信息技术管理部`

*   负责根据总行运营管理部门提出的业务需求，完成统一对账系统的开发，优化等工作

`一级分行运营管理部` --- 本分行银企对账工作的归口管理部门

*    负责牵头落实总行银企对账管理规定

*    负责集中打印、封装、发送对账单，集中催收、回收、审核对账回执或回签记录，集中处理不符信息等

*    负责组织对连续不对账客户账户进行支付监管

*    负责对本分行银企对账工作进行培训、统计分析、监督、检查等

一级、二级分行公司银行部、金融同业部等单位客户管理部门

`一级、二级分行公司银行部、金融同业部等单位客户管理部门`

- 负责向客户宣传银企对账工作
- 负责协调组织相关客户经理与一级、二级分行运营管理部人员共同进行上门对账

`支行`

- 负责与客户签订银企对账协议、维护与变更客户联系信息、为客户开通与变更同意对账系统功能等工作

###  4、银企对账范围 :bookmark_tabs: 

- 单位客户在我行开立的各类存款、贷款、保函、银行承兑汇票账户，但下列账户不属于此范围：
  - 1. 长期不动户

  - 2. 临时验资户
  - 3. 财政零余额
  - 4. 委托贷款基金账户
  - 5. 银团贷款、联合贷款参加绗贷款账户
  - 6. 五级分类中次级、可疑、损失类的贷款账户
  - 7. 被有权机关司法冻结、被我行支付监管的账户

### 5、应对账客户 :bookmark_tabs:

应对账客户：在一级分行辖内以客户为单位进行，凡开立有银行对账范围内账户的客户均为应对账客户

### 7、银企对账频率 :bookmark_tabs: 

- 银企对账按月实施，每个自然月未一个账期（包括电子对账和纸质对账）


### 8、银企对账方式 :bookmark_tabs: 

|          |                                                              |
| :------: | :----------------------------------------------------------- |
| 电子对账 | 我行统一对账系统向客户发送电子余额对账单，由客户在同一对账系统中确认并反馈我行的银企对账方式 |
| 纸质对账 | 我行通过统一对账系统打印纸质余额对账单并发送给客户，由客户核对后加盖预留印鉴或公章进行确认并反馈我行的银企对账方式，纸质对账分为邮递对账和上门对账，其中：**1. 邮递对账**：指通过邮递将纸质余额对账单发送给客户的纸质对账方式，**2. 上门对账**：指我行人员主动上门将纸质余额对账单送达客户的纸质对账方式 |

### 9、上门对账 :bookmark_tabs:

- A类B类客户
  - 每个自然季度内至少回签一次，如自然季度内前两个账期均未回签，则第三个账期分行应对该客户进行上门对账
- 可疑客户
  - 对于日常管理中发现的其他可疑账户，各行应及时对相关客户进行上门对账

### 13、电子对账流程 :bookmark_tabs:

> 1、开通系统功能并建立操作员
>
> - 客户选择电子对账方式的，应填写《申请表》，经<font color=red>签约管理行审核无误</font>后，为其开通统一对账系统功能并建立操作员
>
> 2、电子余额对账单发送
>
> - 统一对账系统于 <font color=red>每月初两个工作日内</font>自动生成电子余额对账单，并发送给客户进行对账
>
> 3、监测催收
>
> - 银企对账管理人员应每日检查统一对账系统的客户电子对账回签情况。电子余额对账单 <font color=red>发出两周后</font>未收到客户回签信息的，银企对账管理人员应通过电话、短信等方式督促客户进行对账
>
> 4、回签处理
>
> - 对于客户反馈的未达账项等不符信息，银企对账管理人员应于 <font color=red>三个工作日内</font>完成拆线呢与确认，设计账务调整的，应于 <font color=red>五个工作日内</font>完成

### 16、银企对账信息管理 :bookmark_tabs:

- 银企对账信息包括客户联系信息、对账记录信息

  - 客户联系信息：客户联系人、联系电话、联系地址、邮政编码等
  - 对账记录信息：电子对账信息、纸质余额对账单回执等

  

签约管理行负责客户联系信息的维护与更新

对账记录应按照我行会计档案的管理要求进行保管，其中：

  （一）电子对账信息由总行统一管理，按年存储备份。

  （二）纸质余额对账单回执等纸质对账记录由一级分行按照客户号、对账期间整理装订，并按会计档案管理办法相关规定进行归档保管。



### 17、银企对账外包管理 :bookmark_tabs:

各行可结合银企对账工作实际情况，在风险可控的前提下将余额对账单 <font color=red>发送、催收、回收环节</font>委托给行外外包服务供应商（如专业服务公司、快递公司等）进行处理<font color=red>上门对账不得进行外包</font>



### 18、银企对账考核 :bookmark_tabs:

总行将<font color=red>电子对账签约率及A类、B类、C类客户回签率</font>纳入运营工作考核范围，定期对分行进行考核

- 电子对账签约率需要 *≥* 80%
- A类 *≥* 95%
- B类 *≥* 80%
- C类 *≥* 70%






> - 电子对账及邮递对账的余额对账单回收期为<font color="red">两个月</font>，回收期内如遇国庆、春节等长假，回收期可延长10个工作日，上门对账应于<font color="red">当季度内</font>完成
>
> - A类、B类客户在<font color="red">每个自然季度</font>内至少回签一次，C类客户<font color="red">每半年</font>内至少回签一次





### 19、银企对账检查 :bookmark_tabs:

分行运营管理部负责人<font color="red">每季度</font>检查一次

分行运营管理的分管行长<font color="red">每半年</font>检查一次

检查内容：银企对账检查应对银企对账管理及上门对账人员配备和履职情况、对账组织和对账单回签情况、<font color="red">电子对账回签IP地址情况</font>、不符信息处理情况、客户联系信息质量、对账外包工作情况等进行检查。



## 二. 中信银行股份有限公司合规风险管理政策（4.0版，2018年）

### 1、总则:bookmark_tabs:

- 性质：纲领性文件、基本制度、重要宣言

- **基本准则**：依法合规、稳健经营

  > 本行生存发展的生命线，本行始终恪守的基本准则。




- **风险合规管理**
  - 概念
        - 贯彻落实 **金融回归本源**、**服务实体经济**、**供给侧结构性改革**、**打赢防范金融风险攻坚战** 等国家政策要求。
        - 实现 **全面合规**、**有效合规**、**持续合规**。
    - 遵循原则
      - 加强党的领导原则
      - 全员合规原则
      - 可持续原则
      - 独立履职原则
    - 内容
      - 本行全面风险管理体系：合规风险、信用风险、市场风险、操作风险。
      - 合规是通过立规方式，设立底线，防范各类可计量风险的最低要求。
      - 接受中国银保监会及其他有权机关监督、检查、评价。

### 2、合规理念和合规文化:bookmark_tabs:

- **经营理念**：平安中信、合规经营、科技立行、服务实体、市场导向、创造价值

- **管理目标**：有担当、温度、特色、尊严的最佳综合金融服务企业。

- **合规理念**：合规创造价值

- **合规意识**：依法合规经营人人有责

- **合规氛围**：高管引领，全员参与

- **行为规范**：合规底线，主动合规

- **合规文化**：合规文化是本行企业文化的核心组成部分，本行致力于规范和培育富有中信银行特色的合规文化，助力本行全面建设“平安中信”。

- 全面建设"平安中信"的方式

  - 将风险合规文化建设与企业文化建设和党建相融合，培育全员尊崇规章、坚守风险底线的规矩意识和风险理念
  - 完善员工全面考核评价机制
  - 强化问责
  - 加强典型风险案例警示教育
  - 强化员工行为管控

  

- 法律法规以及监管明令禁止事项 --> 严禁各级机构和员工逾越监管红线。

- 无监管政策、监管规定不明确情形 --> `实质重于形式` 和 `商业银行审慎经营` 原则，进行合规合法的全面审核。

### 3、合规风险管理组织架构及职责体系:bookmark_tabs:

**组织架构**

- **组织架构**：**董事会、监事会、高级管理层、总分行合规管理职能部门、内部审计部门、其他业务条线管理部门**
  - 董事会：决策机构，对本行经营活动合规性负有最终责任。
  - 监事会：合规管理监督职责
  - 高级管理层：管理本行的合规风险，定期向监事会通报情况。
- 合规风险管理 `三道防线`：
  - 一道：业务经营机构和业务管理部门
  - 二道：合规部门与各类风险管理部门
  - 三道：内部审计部门、纪检监察部门

**职责体系**

- 合规负责人及合规部门
  - 合规部门权限：
    - 相关部门和员工在提供信息方面有合作义务。
    - 有权独立调查内部违规事件；有权按照合规风险报告线路报告。
    - 各级机构（部门）对合规部门开发调查、检查、督察等应主动给予配合、支持。
- 合规负责人：由分管合规工作的行领导担任。
- 合规部门负责人：本行各级机构（部门）中负责合规管理工作的专业管理人员。

### 4、合规风险管理机制和监督机制:bookmark_tabs:

**管理机制**

- 外规内化工作机制
- 合规审核制度：对新产品、新业务、新制度进行合规性审核
- 合规风险评估制度
- 统一的检查发现问题整改制度
- 重大内控合规风险事件报告制度

**监督制度**

- 合规考核：全面覆盖、公平、公正、公开、注重实质、正向激励、动态管理
- 合规问责：违规必究、奖惩分明、以事实为依据、以规则为准绳、惩罚与教育相结合
- 诚信举报：遵循实事求是、回避、保密原则，建立诚信举报机制

**报告体系**

遵循全面性、真实性、及时性、完整性的原则建立明确的合规风险报告路线，主要包括：

- 高级管理层向董事会、监事会的报告
- 总行合规部向高级管理层的报告
- 兼职合规人员向合规部门报告
- 一级分行、二级分行合规部门向所属机构负责人和上级合规部门的报告

### 5、合规资源保障机制:bookmark_tabs:

- 本行各级管理层 --> 合规部门完成风险管理计划 --> `人力`、`财力`、`技术` 支持保障
- 各级合规部门开展合规风险识别、评估、监测工作 --> `IT 系统` <-- 信息科技部门给予支持和保障

### 6、境外分行、附属机构合规管理:bookmark_tabs:

  - 遵守所在地法律法规、监管规则；关注境外资源、劳务用工、宗教及文化习俗。
  - 遵守我国以及所在国家、地区所适用的法律、规则和准则。

## 三、We合规第四期-员工禁止行为规定

- 行为管理：违法行为、违规行为、不当行为、异常行为

- 《员工违规行为处理办法》-- `业务角度`

  - 员工十个严禁

    - 民间借贷、非法集资

    - 黄赌毒黑恐

    - 上班时间个人投资

    - 经商办企

## 四、《中信银行柜面业务操作人员“十禁”规定“》

### 柜面业务操作人员

- 不按规定查验客户身份、违反客户意愿、无凭无据进行业务操作，冒充或代替客户签字或输密，伪造、变造会计凭证，或对明知是违规办理的业务不抵制、不报告。
- 违纪办理支票、银行汇票等 `支付结算业务`。
- 私自 `代客` 保管银行卡、票据、印鉴、网银USBKEY（易企对证书）等重要物品，或代客户进行银企对账、手机银行、网上银行、智慧柜台操作。
- 未履行交接手续，将本人保管的凭证、印章、钥匙、密码等重要物品交与他人使用，或未经审批将 `凭证/印章带离营业场所`。
- 将 `系统用户` 借给他人使用，或在未锁屏/签退系统，未妥善保管好现金、凭证、印章等重要物品的情况下离柜、离岗。

- 办理 `本人` 业务，或在本网点 `代理他人办理开户` 业务。
- `逆流程` 办理柜面业务，或办理 `未完成审批授权` 业务。
- 携带与 `工作无关物品` 进入 `现金区`。
- 篡改、隐匿、故意损毁会计档案，或 `泄露` 客户、银行会计信息及反洗钱工作信息。

### 柜面业务授权人员

- 不按照规章制度要求进行授权。
- 越级授权、逆向授权、超范围授权。
- 为与实际业务不符的事项进行授权，或授权时弄虚作假。
- 发现违法、违规、违纪行为而不及时制止、纠正，继续办理授权。
- 对授权事项随意积压、延误处理。
- 现场授权人员未对客户身份、业务处理真实性等进行核实即进行授权。
- 未审查业务影像资料、业务现场审核意见、交易要素等即进行非现场授权。
- 未经批准或未办理相关手续，即对审批授权事项进行转授权。
- 同一授权事项多次进行转授权。
- 超规定期限进行转授权。

### 会计经理

- 不按规定正确组织开展会计核算工作，伪造、变造会计凭证、会计账簿或提供虚假会计信息。

- 单位结算账户开户、变更、撤销审核或预留印鉴 `审核流于形式`。

- 不按规定办理客户账户的查冻划。

  > 冻： 冻结、解冻

- 不按规定组织对现金、重要空白凭证、账务等进行定期核对。

- 不按规定办理运营事项交接或不认真履行监交职责。

- `泄露` 客户、银行会计信息及反洗钱工作信息。

- 违背 `独立性原则`，按他人授意违规办理业务，或明知违规仍同意继续办理。

- 授意、指使、强令柜员违规操作，或明知柜员违规操作但不制止、不纠正。 -- `明知故犯`

## 五、《中信银行代保管物品管理办法（2.0版，2017年）》 

> 中信银行代保管物品管理办法 2.0 (2017年)

- 代保管物品：指除抵、质押品以外，由保管部门代委托部门保管的 `有价物品` 及 `重要物品`。

  > 保管部门：支行运营管理部门
  > 委托部门：分、支行、公司银行部、金融同业部、零售银行部

- 代保管物品必须严格管理，除专项业务（票据类）有特定要求，原则上以 `封包形式` 放入运营管理部 `	库房` 或者 `保险柜` 保管，并建立 `代保管物品登记簿`。

- 代保管物品封包遵循 `谁委托、谁封包、谁交存`，`谁委托、谁领取` 的原则。

  - 1 物品 1 封包
  - n 物品 1 封包

- 代保管物品在 "606 代保管重要物品" 科目下，按品种设专户进行核算。1 封包 1 RMB 标准记账。

- 分支行委托部门负责办理本机构代保管物品出入库，确保拟入库保管物品的真实性，并配合运行管理部门做好代保管物品核对工作。

  - 入库

    - 委托部门经办人：填 `代保管物品出 (入) 库单` --> 一式二联，换人复核，双人共同核 --> 代保管物品封包并在骑缝处签章 --> 委托部门有权人签章 --> 前述经办人、复核人交至保管部门
    - 保管人员：检查封包完好；出(入)库单签章完整 --> 核对出 (入) 库单要素无误 --> 出 (入) 库单填写封包编号 --> 登记 `代保管物品登记簿` --> 委托部门双人在登记簿签章完成入库手续

  - 出库

    - 委托部门经办人：填 `代保管物品出(入)库单` --> 委托部门有权人签章 --> 经办复核人至保管部门办理出库手续

    - 委托部门经办人：双方同检查封包完整签章完整 --> 保管人员登记 `代保管物品登记簿` --> 委托部门领取人双人在登记簿签章签 (第一联由保管部门随当日传票；第二联退委托部门) 

      > 代保管物品的交接工作应全程在监控下进行。

- 代保管物品的保管

  - 各行根据实际情况在金库、重要凭证库房或保险柜中保管代保管物品，并确保该区域与对外营业区进行有效隔离，符合 7*24 小时保护要求。
  - 代保管物品由双人保管，封包必须分类、顺序排放。
  - 代保管物品原封完好，封包内物品真实性、完整性、合规性由办理入库的委托部门负责；若封包破损，封包内保管物品发生多、缺由保管部门负责。
  - 有权机关调查代保管物品 --> 封包进行拆封检查 --> 由委托部门办理出库手续 --> 检查结束后重新封包入库。

- 代保管物品查库

  | 人员                                   | 频率           | 代保管物品   |
  | :------------------------------------- | :------------- | :----------- |
  | 分行库管主管、支行会计经理             | 每月  *≥*  1   | 全面盘查     |
  | 分行运营管理部负责人、支行分管运营行长 | 每半年  *≥*  1 | 监视查库     |
  | 分行运营条线行长、支行长               | 每年 *≥*1      | 监视查库     |
  | 分行运行管理部门                       | 定期或不定期   | 检查辖属机构 |

- 保管部门，于每季度初 10 日内，将保管物品清单封包清单通过书面/电子公文形式，发至委托部门确认核对。

  - 委托部门应于当月月末之前反馈确认结果，60 天内未予反馈视为核对一致。
  - 每次核对完毕，核对资料专夹保管，并登记 `定期核对登记簿`。
  - 核对库存实物与登记簿不符，发生丢失、损毁，应查明原因，并按柜面业务运营重大事项报告相关规定进行报告。

- 运营条线的业务办理过程中产生的需要暂存的重要物品，应比照代保管物品进行管理。

  - 此类物品按照项目/类别管理，可不进行封包。
  - 此类物品办理出入库时可不填写 `代保管物品出 (入) 库单`，但应由经办人与代保管物品保管人员办理交接，并登记 `代保管物品登记簿` (设有专用登记簿的除外)。

## 七、现金出纳业务管理基本规定 

- 现金运营指本/外币现金实物的
  - 保管：限额管理、合理压缩库存
  - 调缴：分 <----> 支
  - 清分：全额清分
  - 收付：现金收入先收款后记账；现金付出先记账后付款
  - 兑换：现金兑换先记账后兑付

### 业务管理

#### 现金库房管理

- 现金库房：金库和日间库
- 双人管库
- “五同”、“六防”；监控无死角；密码钥匙直接交接，交接时必须变更

#### 现金调缴管理

- 本行与人行或同业银行间：金库经理、运管部负责人审批后申请
- 营业机构与分行间：会计经理审批后申请
  - 库管柜员申请
  - 紧急业务柜员间现金移交，由会计经理审批同意并监交办理

#### 现金清分管理

- 现金清分管理，指回笼的现金进行面额和套别区分、真假币鉴别、数量统计、质量分类的处理过程。包括机械清分和人工清分。
- 对 `钞币` 进行逐张清点、识伪、挑残、扎把、签章、打捆
- 对 `外付出现金` 应 `全额清分` 并记录 `冠字号码`

#### 柜面现金运营业务管理

- 柜面现金收入
- 柜面现金付出
- 零钞兑换
- 残损钞兑换
- 柜员尾箱管理
  - 每日营业时，以柜员为单位将现金存放于尾箱、带锁抽屉或者保险柜中保管，确保随启随闭
  - 每日营业时，现金柜员离开岗位时，必须将现金缩入尾箱、抽屉或保险柜
  - 每日营业结束，尾箱内现金换人清点核对，尾箱加锁或加封
  - 营业机构不得将现金（自助设备除外）存放在本机构过夜，统一放入金库或者寄库

#### 反假和样币管理

- 假币收缴

  - 两名具有反假货币相关资格的柜员在监控下当面予以收缴，并向客户出具《假币收缴凭证》，假币不得再退还客户。
  - 营业结束后，将假币交库管柜员放入尾箱或者保险柜保管。

- 节假日收缴的假币：柜员随现金入尾箱保管，并于节假日后第一个工作日营业结束前，将假币交库管柜员保管。

- 营业机构收缴的假币：定期上交分行，由其统一向当地人民银行解缴。

- 样币管理：样币禁止流通，流入市场的予以收回。

- 收缴假币与公安部门的联系

  |   立即报告当地公安部门    |                     当日报告当地公安部门                     |
  | :-----------------------: | :----------------------------------------------------------: |
  | 一次性假人民币$\geq500$元 | 1）一次性假人民币$\geq200$元</br>2）一次性假外币$\geq10$张（枚）</br>3）持有人不配合我行收缴行为</br>4）新造假手段制造的假币</br>5）重大假币案和新类型假币</br>6）制造贩卖假币线索 |

#### 现金处理机具管理

- 现金处理机具：办理现金运营业务所需的专用设备，包括清分机、点钞机、复点机、扎把机、捆钞机、外币鉴别仪（点钞机）

#### 现金出纳差错管理

- 查清原因的 --> 会计经理或者金库经理审批后处理

- 当日无法处理的 --> 挂账处理

- 无法查清原因且挂账满一年的 --> 年终决算前按照权限由有权人审批同意后计入 `营业外收支`

  > 发现现金出纳差错，会计经理或者金库经理应立即组织查找，严禁长款隐匿，短款自补。

## 八、单位定期存款账户相关规定

### 基本概念

- 单位定期存款账户：各级财政金库、机关、企业、事业单位、社会团队、部门等机构，资金存入银行，并事先约定存期、利率，到期支付本息的一种存款方式。

### 账户分类

- 开户主体属性
  - 一般单位定期存款账户
  - 同业定期存款账户
- 期限
  - 标准期限：1月、3月、6月、1年
  - 非标准期限

### 业务类型

> 主要针对一般单位定期存款账户业务讲解。

- 账户开立
- 存续期间对账
- 提前支取
- 到期结清

### 风险控制

- 开户
  - 存款人提交申请材料
  - 柜台人员操作
  - 会计经理复审
  - 印鉴预留
- 支取
  - 存款人操作
  - 办理支取

#### 开户

- 提交资料

  - 我行转开方式

    - 填写开户申请书
    - 存款人签发、付款账户为存款人已开立结算账户的支票/划款凭证，付款金额与拟开立定期存款金额一致
    - 预留印鉴

  - 他行转开方式

    - 填写开户申请书

    - 提交营业执照、法人身份证、授权委托书开户资料

    - 存款人应办理定期存款开户手续前，通过他行同名账户将存款资金转入我行 

      > 财政存款等有特殊要求的从其规定。

    - 预留印鉴

      - 单独留存定期存款账户预留印鉴
      - 所在一级分行分割内已开立的账户公用预留印鉴

- 柜台人员操作

  - 开户申请书 -- 内容完整
  - 公章一致
  - 监控名单
  - 开户资料
  - 存款金额 -- 我行结算账户可用余额 $\geq$ 拟开立定期存款金额
  - 开户核实 -- 他行转开方式 -- 电话核实法人/单位负责人/财务负责人的开户意愿

- 会计经理复审

  - 资料审核无误，会计经理复核。提交定期存款开户交易，交易成功打印 `存款证实书`
  - 存款证实书（第一联）--> 加盖柜面业务专用章；并建电子底卡

#### 支取

- 全部支取（定期存款结清）
  - `Case.01`：柜面人员 --> 打印业务凭证及利息清单 --> 存款人
  - `Case.02`：`他行转开方式` -- 填写特种转账品凭证（两联），按原留存资金划入凭证所载路径将存款资金划回
- 部分支取
  - `Case.01`：留存金额高于定期起存金额 -- 柜台人员提交定期存款部分提前支取交易，按原存款开户日期、期限、利率、留存金额打印新证实书，同时打印通用凭证及利息清单
  - `Case.02`：`他行转开方式` -- 填写特种转账品凭证（两联），按原留存资金划入凭证所载路径将存款资金划回
- 结算账户变更/注销
  - `Step.01`：存款人出局开户许可证、销户申请书等证明材料
  - `Step.02`：营业机构审核无误，电话核实法人/单位负责人/财务负责人对变更后的划款路径及经办人身份
  - `Step.03`：存款资金划入其基本存款账户
